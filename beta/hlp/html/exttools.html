Внешние утилиты

Закладка External Tools диалогового окна Config / Externals предоставляет пользователю возможность быстрого доступа к необходимым ему приложениям. В поле Caption заносится название приложения, в поле String to execute - командная строка его запуска. После этого указанные в этом диалоговом окне внешние утилиты будут доступны из меню Tool / External Tools.

Вы можете занести в таблицу External Tools до 20 приложений.
