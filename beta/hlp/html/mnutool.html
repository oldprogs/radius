Меню Tool

  • Compile Nodelist - при выборе этого пункта происходит компиляция всех файлов типа Nodelist, которые прописаны в окне конфигурации списка узлов.

  • Browse Nodelist - выбор этого пункта приводит к открытию окна со списком доступных узлов. Список узлов представляется в виде полного дерева с разделением по зонам, регионам, сетям, хабам и узлам. Сеть считается принадлежащей региону тогда, когда первые две цифры ее номера совпадают с номером данного региона. Поле Address Search в окне просмотра списка узлов служит для быстрого перемещения курсора к узлу по указанию его адреса. Переход сопровождается раскрытием соответствующих ветвей. Не стоит злоупотреблять функцией расктытия всего дерева целиком (Shift+GrayAsterisk). Список может содержать несколько тысяч узлов, и на полное его раскрытие может потребоваться не только очень продолжительное время, но и огромное количество системных ресурсов.

  • Browse Nodelist at... - вызывает окно со списком доступных узлов и активизирует элемент списка, соответствующий указанному адресу.

  • External Tools - данное раскрывающееся меню предназначено для запуска списка внешних утилит, опредяемого в сетке ввода меню Config / Externals / External Tools.

  • Edit File Request ... - этот пункт вызовет открытие окна редактирования файлового запроса.

  • Attach Files... - позволяет определить файлы, подлежащие отправке на указанный узел. В окне выбора файлов допускается множественная отметка в пределах одного каталога. Для этого, отмечая файлы, следует воспользоваться клавишами Shift и Ctrl.

  • Create a Flag... - этот пункт полезен для создания прозвонок типа Outb., то есть инициированных буфером отправки. Так как прозвонки типа Manual (созданные нажатием F5) игнорируют время работы станции, необходим механизм создания прозвонок, учитывающих это время. Create a Flag (Ctrl+F8) как раз и является таким механизмом. По сути, этот пункт создает в буфере отправки пустой ?LO- или CLB-файл. Напомним, что время работы определяется флагами CM/Txy в списке узлов или сетке переопределений. Пункт Create a Flag сохраняет возможность создавать ручные прозвонки (Manual Poll). Для этого в списке “Attach Status” выберите “Crash&Poll”.

  • Outbound SmartMenu - вызов Outbound SmartMenu, который возможен также нажатием правой кнопки мыши на элементе буфера. Outbound SmartMenu является мощнейшим инструментом для работы с элементами буфера. В специальной главе описаны возможности этого инструмента и принципы работы с ним.
