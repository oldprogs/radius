Программный интерфейс Радиуса

Программный интерфейс (Application Programming Interface, API) Радиуса - это набор предоставляемых программой функций, доступных для вызова из внешних приложений. Он обеспечивает возможность создания собственных программных средств управления почтово-транспортной системой.

В разработке API Радиуса и использующих его внешних приложений участвует Eugene Palenock. В качестве примера такого приложения можно привести программу, принимающую данные от внешнего АОНа и передающую управление модемом Радиусу в соответствии с определенной программной логикой. В результате появляется возможность автоматического различения входящих вызовов без использования комплекса программ, взаимодействующих друг с другом посредством TAPI. В целом же разработка API Радиуса имеет своей целью обеспечение фактически неограниченной гибкости управления системой.

Более подробное описание API Радиуса, примеры его использования и готовые программы можно найти на Интернет-сайте, который поддерживает Eugene Palenock.
