Трей Windows

Параметр Описание
Always in tray Определяет, будет ли значок Радиуса виден в трее Windows, когда окно Радиуса не минимизировано.
Stealth mode Невидимый режим. Держать Радиус свернутым, без звуков, без иконки в трее и без кнопки в панели задач. В невидимом режиме окно Радиуса активизируется сочетанием клавиш, заданным в поле HotKey to popup from stealth mode.
Show baloon tooltip Показывать "облачко" с краткой информацией о соединении с удаленным узлом во время сеанса связи над значком Радиуса в системном трее. Работает только под Windows 2000/XP.
Show baloon tooltip when minimized Показывать Balloon Tooltip, когда Радиус минимизирован в трей. Если этот флажок снят, Balloon Tooltip показывается только если Радиус развернут и установлена опция Always in Tray. Работает только под Windows 2000/XP.
