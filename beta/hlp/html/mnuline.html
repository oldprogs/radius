Меню Line

  • Dial-up Lines - этот пункт служит для открытия окна линии почтовой системы, в раскрывающемся подменю будет предложено выбрать конкретную линию, окно которой Вы желаете открыть.

  • TCP/IP Daemon - этот пункт служит для запуска TCP/IP Daemon.

  • Terminal - запустить терминал.

  • Reset - производит сброс и инициализацию отображаемой линии почтовой системы.

  • Answer call - производит ответ на входящий звонок. Полезно при запрещенном автоответе.

  • Send Modem Commands - посылает команду модему. Этот пункт меню доступен только если Радиус занимает COM-порт.

  • Flush Timeout - при выборе этого пункта происходит принудительное окончание ожидания прозвонки или соединения, и отображаемая линия соответственно переходит в режим набора или ожидания прозвонки.

  • Add to Timeout - при выборе этого пункта к значению времени ожидания прозвонки или соединения отображаемой линии прибавляется 30 сек.

  • Skip File - пропустить текущий принимаемый файл и отложить до следующего сеанса. Эта команда может неоднозначно трактоваться некоторыми системами, поэтому проявляйте особую осторожность при использовании этой команды. См. также Дополнительные параметры прозвонок.

  • Reject File - пропустить текущий принимаемый файл и отказаться от его приема в дальнейшем. Эта команда правильно воспринимается далеко не всеми системами. См. также Дополнительные параметры прозвонок.

  • Open Chat - активизировать сеанс чата.

  • Close - производит закрытие отображаемой линии почтовой системы.

См. также Окно линии почтовой системы
