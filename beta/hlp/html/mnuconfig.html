Меню Config

  • Master Password - эти пункиы позволяют защитить данные конфигурации посредством Master Password

  • Start-up - этот пункт вызывает диалоговое окно конфигурации параметров запуска

  • Paths - этот пункт вызывает диалоговое окно настройки каталогов

  • Nodelist - этот пункт вызывает диалоговое окно конфигурации списка узлов

  • Passwords - этот пункт вызывает диалоговое окно конфигурации паролей

  • Preferences - в диалоговом окне "Настройки" задаются установки элементов визуального оформления Радиуса: язык интерфейса, цвета и т.д., - а также дополнительные параметры

  • Dial-up - этот пункт вызывает диалоговое окно конфигурации модемных соединений

  • TCP/IP Daemon - этот пункт вызывает диалоговое окно конфигурации TCP/IP Daemon

  • File-boxes - этот пункт вызывает диалоговое окно конфигурации файл-боксов

  • Polls - этот пункт вызывает диалоговое окно конфигурации параметров прозвонок

  • File Requests - этот пункт вызывает диалоговое окно конфигурации файловых запросов

  • Externals - этот пункт вызывает диалоговое окно конфигурации внешних приложений

  • Events - этот пункт вызывает диалоговое окно конфигурации событий

  • Encrypted Links - этот пункт вызывает диалоговое окно конфигурации адресатов шифрованного сеанса связи BinkP

  • Node Inspector - этот пункт вызывает диалоговое окно Инспектор Узлов
