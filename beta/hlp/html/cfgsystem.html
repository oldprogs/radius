Системные параметры

Не вносите изменений в данной секции, если вы не уверены в их корректности!

Параметр Описание Пример
Mutex name Изменив значения этих параметров, вы сможете запустить новую копию Радиуса. Будьте осторожны, не меняйте эти значения, если вы не знаете, что это такое! ARGUS_SEMAPHORE
Activate Event Name ARGUS_EVENT_ACTIVATE
Directory Watcher Event Name ARGUS_EVENT_DIRECTORY_WATCHER
EMSI CR Возможность посылки определенных символов в качестве терминатора EMSI-последовательности (символ CR или пара символов CR LF). %0D
%0D%0A
Simple BSY Возможность открытия .BSY-файлов без DELETE_ON_CLOSE. В этом случае файлы будут удаляться командой DeleteFile. Это может быть полезно на сетевых устройствах.  
Close BWZ Возможность держать закрытым файл badwazoo.lst. Hе изменяйте этот файл, когда запущен Радиус. Ваши изменения могут быть перезаписаны.  
Ignore end session Возможность игнорировать сигналы завершения сессии (WM_ENDSESSION и WM_QUERYENDSESSION). Это может быть полезно при работе сервисом NT.  
Ignore BadWaZoo Size Игнорировать информацию о полном размере недопринятых файлов и пакетов, помещенных во временный каталог. Это может быть полезно, например, в нештатных ситуациях, когда при обрыве связи по какой-либо ошибке полный размер файла был записан неверно, и файл требуется допринять по имени. См. также Панель BadWaZOO.  
Force fax page Запись частично принятых страниц факса.  
Increment arcmail Автоматическое увеличение последнего символа расширения принятых почтовых пакетов (ARСmail), если файл с таким именем уже присутствует в каталоге приема.  
Disable winsock traps Возможность избежать перезагрузки Радиуса при возникновении фатальных ошибок WinSock (shutdown/closesocket errors). Использовать эту опцию не рекомендуется, так как нормальная работа после фатальных ошибок маловероятна.  
Winsock version Возможность выбора версии WinSocket:
WinSocket v1,
WinSocket v2,
автоматический выбор.		 Winsock auto
Winsock v1
Winsock v2
