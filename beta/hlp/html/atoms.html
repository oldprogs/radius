Атомы

Атом - это частица события, влияющая на один параметр конкретной линии, к которой событие привязано.

Поддерживаются следующие атомы:

Accept BPS Efficiency Min
(Допуск Эффективность Минимальная)		 Минимальная производительность в процентах на входящих соединениях. Если во время действия этого атома значение BPS входящего соединения окажется меньше указанного, произойдет отсоединение.
Accept Callback Enable
(Разрешить Callback Запросы)		 Разрешить обратный вызов. Во время действия этого атома Радиус будет обрабатывать запросы обратного вызова.
Accept CPS Value Min
(Допуск CPS минимальный)		 Минимальная скорость передачи на входящих соединениях. Если во время действия этого атома значение CPS входящего соединения окажется меньше указанного, произойдет отсоединение.
Accept CRAM Disabled
(Допуск CRAM запретить)		 Запретить инициирование механизма CRAM-MD5 на входящих соединениях.
Accept Dummy-Z Disabled
(Допуск Dummy-Z запретить)		 Запретить передачу пустого пакета на входящих соединениях на ZModem. Если этот атом не активен (по умолчанию), в случае отсутствия файлов для передачи, на протоколе ZModem будет передан пустой PKT-файл. Передача такого файла является мерой обеспечения нормального проведения сеанса ZModem.
Accept Duration Max
(Допуск Длительность Максимальная)		 Максимальная продолжительность входящего сеанса связи в минутах. После истечения указанного интервала времени произойдет отсоединение.
Accept EMSI Disabled
(Допуск EMSI запретить)		 Запретить использование протокола EMSI на входящих соединениях.
Accept Files Required / Accept Files Delayed
(Допуск Файлов Требуемые / Допуск Файлов Отложенные)		 Файлы, прием которых возможен / Файлы, прием которых следует отложить. Чаще всего используются для запрещения в ZMH приема файлов,за исключением FTSC-0001-пакетов. Как и в настройках обработчиков, здесь действует макрос %ARCMAIL%.
Accept Files Refuse Delete
(Допуск Файлов Запрещенные Отказ)		 Файлы, удаляемые без приема. Указанные в этом атоме файлы будут безвозвратно удалены без приема. Как и в настройках обработчиков, здесь действует макрос %ARCMAIL%.
Accept FReq Count Public
(Допуск Запросов Файлов Непарольных)		 Ограничение запрашиваемых файлов по числу для непарольных сеансов связи
Accept FReq Count Pwd-Prot
(Допуск Запросов Файлов Парольных)		 Ограничение запрашиваемых файлов по числу для парольных сеансов связи
Accept FReq Disable
(Запретить Файловые Запросы)		 Запретить отработку файловых запросов.
Accept FReq Duration Public
(Допуск Запросов Длительность Непарольных)		 Ограничение запрашиваемых файлов по длительности сеанса связи для непарольных сеансов связи
Accept FReq Duration Pwd-Prot
(Допуск Запросов Длительность Парольных)		 Ограничение запрашиваемых файлов по длительности сеанса связи для парольных сеансов связи
Accept FReq Size Public
(Допуск Запросов Size Непарольных)		 Ограничение запрашиваемых файлов по суммарному размеру для непарольных сеансов связи
Accept FReq Size Pwd-Prot
(Допуск Запросов Объем Парольных)		 Ограничение запрашиваемых файлов по суммарному размеру для парольных сеансов связи
Accept FTS-0001 Disabled
(Допуск FTS-0001 запретить)		 Запретить использование протокола FTS-0001 на входящих соединениях.
Accept Hydra Disabled
(Допуск Hydra запретить)		 Запретить использование протокола Hydra на входящих соединениях.
Accept Link Required / Accept Link Forbidden
(Допуск Протоколов Требуемые / Допуск Протоколов Запрещенные)		 Протоколы соединения, допустимые к приему / запрещенные для приема. Используется для запрещения соединения без коррекции ошибок в определенное время. Протоколы перечисляются через пробел (например: HST ARQ).
Accept Niagara Disabled
(Допуск Niagara запретить)		 Запретить использование протокола Niagara на входящих соединениях. Niagara используется протоколом BinkP (и шифрованным BinkP) на модемных соединениях.
Accept No Incoming
(Допуск Входящие запретить)		 Не принимать входящие соединения. Во время активности этого атома коммуникационный порт используется только для исходящих звонков. В остальное время порт доступен другим приложениям.
Accept Nodes Only Listed
(Прием только известных)		 Допускать к приему только узлы, прописанные в нодлисте, в паролях или переопределениях. Если во время действия этого атома иной узел попытается установить сеанс связи, произойдет отсоединение.
Accept Nodes Only Pwd-Prot
(Допуск Узлов Только-Парольных)		 Допускать к приему только парольные узлы. Если во время действия этого атома непарольный узел попытается установить сеанс связи, произойдет отсоединение.
Accept Nodes Required / Accept Nodes Forbidden
(Допуск Узлов Требуемые / Допуск Узлов Запрещенные)		 Схемы ограничений для входящих сессий: требуемые условия / условия запрета. Атомы могут использоваться на крупных узлах для распределения суточной нагрузки. Эти атомы допускают использование масок. См. также Схемы ограничения прозвонок.
Accept Speed Min
(Допуск Минимальная Скорость)		 Минимально допустимая скорость входящих соединений. Если во время действия этого атома скорость входящего соединения окажется меньше указанной, произойдет отсоединение.
Accept ZModem Disabled
(Допуск ZModem запретить)		 Запретить использование протокола ZModemна входящих соединениях.
Accept YooHoo Disabled
(Допуск YooHoo запретить)		 Запретить использование протокола YooHoo на входящих соединениях.
Chat Bell
(Сигнал диалога)		 Проигрывать звуковой файл при запросе чата удаленной стороной. Если в поле "File Name" не указан звуковой файл, звукового оповещения не будет. Этот атом перекрывает установки диалогового окна "Настройки" (параметр Chat Bell секции Main).
Del Old *.BSY
(Удалить старые *.BSY)		 Удалить старые файлы BSY. См. также параметр Max BSY Age в секции Main диалогового окна "Настройки".
Disable Chat / Enable Chat
(Запретить диалог / Разрешить диалог)		 Запретить / разрешить чат для указанных адресов.. В поле "Nodes List" допускаются маски. Эти атомы перекрывают установки диалогового окна "Настройки" (флажок Chat Enabled секции Main).
Door
(Сервис)		 Дополнительное приложение-сервис. Добавляет дополнительный сервис, который будет использоваться в дополнение к указанным в Config/External/Doors.
Input Log Format *
(Форматирование Данных)		 Форматирование данных. Этот атом сканирует входные данные и при соответствии регулярному выражению форматирует их и пишет в лог.
Input WatchDog ExtApp
(Сканер - Внешнее приложение)		 Сканер - Внешнее приложение. Атом сканирует пришедшие данные и при соответствии их регулярному выражению запускает внешнее приложение.
Input WatchDog Reset
(Сканер - Сброс)		 Сканер - сброс. Атом сканирует пришедшие данные и при соответствии их регулярному выражению выполняет сброс линии.
Interval Between Polls
(Интервал между прозвонками)		 Период между повторами прозвонки. Атом перекрывает значение Retry seconds, установленное в параметрах прозвонок.
Log EMSI Data
(Хронология EMSI)		 Вести хронологию EMSI-данных. Когда этот атом активен, Радиус записывает данные EMSI в специальный протокол EMSI-данных (r-emsi.log).
Login Script
(Скрипт)		 Запустить скрипт. Этот атом позволяет передавать команды удаленной системе, ожидая ее реакции (например Login и Password на маршрутизаторе) перед началом сессии. Работает только на исходящих сессиях сразу после соединения модемов (Carrier Detected) при совпадении адреса удаленной системы с шаблоном, указанным в строке "Nodes List". В этом списке допустимы маски. В шаблонах могут использоваться регулярные выражения. Дополнительная информация приведена в разделе Скрипты.
Modem Command Answer
(Модем Команда Ответа)		 Командная строка ответа модема на входящий звонок. Обычно используется для установки соединения требуемых протоколов в определенное время.
Modem Command Hangup
(Модем Команда Отсоединения)		 Командная строка модема для отсоединения. Этот атом обычно применяется вместе с атомом Modem Command Init.
Modem Command Info
(Модем Команда Информации)		 Командная строка модема сбора информации.
Modem Command Init
(Модем Команда Инициализации)		 Командная строка инициализации модема. Обычно используется для запрещения включения динамика модема в ночное время, либо для установки определенных протоколов по часовому расписанию.
Modem Command Prefix
(Модем Команда Префикс)		 Префикс командной строки модема набора номера. Обычно используется для установки определенных протоколов для исходящих соединений, а также для работы через внутренние мини-АТС в ночном режиме.
Modem Command Suffix
(Модем Команда Суффикс)		 Суффикс командной строки модема для набора номера.
Modem Disable Reinit
(Запретить регулярную инициализацию модема)		 Запретить регулярную инициализацию модема. Этот атом не разрешает инициализировать модем, находящийся в режиме ожидания.
Modem Error Ext.App.
(Ошибка модема - внешнее приложение)		 Внешнее приложение при ошибке модема. Запускает внешнее приложение, если произошла ошибка инициализации модема.
Modem Script
(Скрипт)		 Модемный скрипт. Этот атом позволяет передавать команды модему, ожидая и обрабатывая его ответы. Атом срабатывает при совпадении ответа модема с шаблоном, указанным в поле "Modem response". В шаблонах могут использоваться регулярные выражения. Дополнительная информация приведена в разделе Скрипты.
Password
(Пароль)		 Дополнительный пароль. Добавляет дополнительный пароль, который будет учитываться в дополнение к указанным в Config/Passwords.
RAS Event
(RAS Событие)		 Соединение посредством RAS. Запуск соединения, настройки которого определены в секции RAS DialUp диалогового окна "Настройки". Данный атом позволяет подключаться к сети по заданному временному графику. Привязывать атом следует к модемным (dial-up) линиям. При этом подключение будет происходить в момент срабатывания события с атомом RAS Event, а отключение - по истечении периода действия события. Опция HangUp when finished предназначена для досрочного отключения от сети, как только очередь отправки окажется пуста. См. также макросы %STARTRAS%, %STOPRAS%
Replace Modem
(Заменить Модем)		 Замена всех установок модема, конфигурируемых в Config /Dialup / Modems. Чаще всего применяется, когда модем обслуживает разные протоколы в зависимости от времени суток. При этом порт остается неизменным.
Replace Restriction
(Заменить Ограничения)		 Замена схемы ограничения прозвонок, конфигурируемой в Config / Dialup / Restrictions. Применяется вместо двух атомов Restriction Required и Restriction Forbidden.
Replace Station
(Заменить Станцию)		 Замена всех установок станции, конфигурируемых в Config / Dialup / Station. Обычно используется на узлах, выполняющих разные функции в разное время.
Response Log Format *
(Форматирование Ответов)		 Форматирование ответов. Этот атом работает так же, как атом Input Log Format, но сканирует не все данные, а только ответы модема.
Restriction Required / Restriction Forbidden
(Ограничения Требуемые / Ограничения Запрещенные)		 Строки необходимых / запрещенных условий прозвонок. Применяются, например, для учета различной тарификации звонков в зависимости от времени суток, а также для учета особенности работы узлов (например, не работающих по праздникам).
Rings
(Звонки)		 Число звонков перед поднятием трубки для ответа модемом. Значение 0 означает запрет поднятия трубки.
Sounds Event
(Звуки)		 Включить / выключить озвучивание событий. Этот атом перекрывает установки секции Main диалогового окна "Настройки" (параметр Play Sounds), включая или выключая (в зависимости от состояния флажка Sounds On/Off) озвучивание событий. Включением сигнала чата управляет отдельный атом Chat Bell.
Traffic Shaper Incoming / Traffic Shaper Outgoing
(Ограничение Трафик Входящий / Ограничение Трафик Исходящий)		 Ограничение скорости передачи для соединений по TCP/IP. Во время действия этих атомов скорость приема / передачи по TCP/IP не будет подниматься выше значения, указанного в поле."BPS for Incoming / Outgoing Traffic". Эти атомы перекрывают установки конфигурации TCP/IP Daemon (параметры Traffic shaper: In / Out).
Transmit BPS Efficiency Min
(Передача Эффективность Максимальная)		 Минимальная производительность в процентах на исходящих соединениях. Если во время действия этого атома значение BPS исходящего соединения окажется меньше указанного, произойдет отсоединение.
Transmit CPS Value Min
(Передача CPS Минимальный)		 Минимальная скорость передачи на исходящих соединениях. Если во время действия этого атома значение CPS исходящего соединения окажется меньше указанного, произойдет отсоединение.
Transmit CRAM Disabled
(Передача CRAM Запретить) 		Запретить инициирование механизма CRAM-MD5 на исходящих соединениях.
Transmit Dummy-Z Disabled
(Передача Dummy-Z Запретить)		 Запретить передачу пустого пакета на исходящих соединениях на ZModem. Если этот атом не активен (по умолчанию), в случае отсутствия файлов для передачи, на протоколе ZModem будет передан пустой PKT-файл. Передача такого файла является мерой обеспечения нормального проведения сеанса ZModem.
Transmit Duration Max
(Передача Длительность Максимальная)		 Максимальная продолжительность исходящего сеанса связи в минутах. После истечения указанного интервала времени произойдет отсоединение.
Transmit EMSI Disabled
(Передача EMSI Запретить)		 Запретить использование протокола EMSI на исходящих соединениях.
Transmit Files Required / Transmit Files Forbidden
(Передача Файлов Требуемые / Передача Файлов Запрещенные)		 Файлы, допустимые к передаче / запрещенные для передачи. Как и в настройках обработчиков, здесь действует макрос %ARCMAIL%. Этот атом не распространяется на файлы с приоритетом Crash, а также на файлы, присоединенные обработчиком файловых запросов.
Transmit Hydra Disabled
(Передача Hydra Запретить)		 Запретить использование протокола Hydra на исходящих соединениях.
Transmit Niagara Disabled
(Передача Niagara Запретить)		 Запретить использование протокола Niagara на исходящих соединениях. Niagara используется протоколом BinkP (и шифрованным BinkP) на модемных соединениях.
Transmit Speed Min
(Передача Скорость Минимальная)		 Минимально допустимая скорость исходящих соединений. Если во время действия этого атома скорость исходящего соединения окажется меньше указанной, произойдет отсоединение.
Transmit YooHoo Disabled
(Передача YooHoo Запретить)		 Запретить использование протокола YooHoo на исходящих соединениях.
Transmit ZModem Disabled
(Передача ZModem Запретить)		 Запретить использование протокола ZModem на исходящих соединениях.
Wait for Connect
(Ожидание коннекта)		 Время ожидания соединения с удаленной системой. Значение, указанное в этом атоме, перекрывает значение Call seconds, установленное в параметрах прозвонок.
ZMH Event
(ZMH событие)		 Включить режим ZMH. Во время ZMH (Zone Mail Hour - почтовый час зоны) разрешается принимать и передавать только прямые почтовые пакеты (см. Буфер отправки системы Binkley). Данный атом перекрывает действие других атомов, разрешающих или запрещающих прием / передачу файлов (Accept Files Required, Transmit Files Required и т.д.)

        ATI2|

Regular expression:

        ZyXEL U-MODEM LINK STATUS REPORT\s*
        Chars Sent *(\w+) *Chars Received *(\w+)\s*
        Octets Sent *(\w+) *Octets Received *(\w+)\s*
        Blocks Sent *(\w+) *Blocks Received *(\w+)\s*
        Blocks Resent *(\w+) *Max Outstanding *(\w+)\s*
        Max Block Size *(\w+) *Retrains Requested *(\w+)\s*
        Link Duration *(\w+) *Retrains Granted *(\w+)\s*
        T401 Timeouts *(\w+) *T402 Timeouts *(\w+)\s*
        FCS Errors *(\w+) *Round Trip Delay *(\w+)\s*
        Xmitter Underrun *(\w+) *Receiver overrun *(\w+)\s*
        Last Speed/Protocol *(.+)\b\s*
        Disconnect Reason *(.+)\b\s*


Format Specifier:

        Was $19 during $11 min, RTT=$16 T
        Tx/Rx: $1/$2 chrs, $3/$4 octs, $5+$7/$6 blks
        Retrains $10/$12, Timeouts $13/$14, FCS Errors $15, Overruns $17/$18
        Disconnect reason: $20
