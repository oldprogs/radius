Библиотека zlib

zlib - это свободно распространяемая универсальная библиотека для сжатия информации без потерь, работающая фактически на любой машине с любой операционной системой. Таким образом, формат данных zlib является платформонезависимым. Авторы zlib - Jean-loup Gailly и Mark Adler. Подробная информация о библиотеке zlib приведена на Интернет-сайте разработчика.

Радиус поддерживает сжатие zlib на протоколах Гидра и BinkP. На данный момент библиотеку zlib, кроме Радиуса, использует только MBSE. Преимущество сжатия zlib особенно заметно при передаче неупакованных прямых почтовых пакетов. При этом, по словам авторов, метод сжатия, примененный в zlib, никогда не приводит к существенному увеличению объема данных, в отличие от LZW.

Для использования сжатия средствами zlib достаточно поместить в Домашний каталог Радиуса библиотечный файл zlib.dll. После этого Радиус будет автоматически определять, поддерживает ли удаленная система сжатие zlib, и по возможности работать со сжатыми исходящим и входящим потоками.

Библиотечный файл zlib.dll поставляется с Пакетом утилит для Радиуса.
