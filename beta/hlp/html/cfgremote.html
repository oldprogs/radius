Удаленное управление

Для удаленного управления системой Радиус предназначена Консоль удаленого управления (Remote Control Console, RCC). Ниже перечислены параметры секции настройки Консоли удаленного управления.

Параметр Описание 
Enable Remote Control Console
 Логический параметр, отвечающий за включение/выключение RCC. 
Crypt RCC password
 Возможность передавать пароль для доступа к RCC в зашифрованном (хешированном) виде. 
Password
 Пароль удаленного доступа к RCC.

Чтобы включить RCC, нyжно установить флажок Enable Remote Control Console и задать пароль доступа к RCC. Доступ к RCC возможен по Telnet, IFC и Dial-up. По BinkP RCC не работает.
