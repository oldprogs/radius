Горячие клавиши

Ниже перечислены горячие клавиши, действующие при запущенном окне Радиуса.

Клавиши Описание Примечание
Esc Сброс и инициализация отображаемой линии Line / Reset
Space Вызвать Outbound Smartmenu на закладке буфера отправки

Окончание ожидания прозвонки или соединения отображаемой линии (при установке соответствующей опции секции Main диалогового окна "Настройки")		 Tool/ Outbound SmartMenu

Line / Flush Timeout
Enter (F2) Перейти в режим изменения значения фокусированного поля Сетки ввода строк
Alt+Ins / Ctrl+Del / Ctrl+GrayPlus Вставить строку / Удалить строку / Добавить строку Сетки ввода строк
Ctrl+Shift+Del (Ctrl+Shift+Y) Удалить сразу все строки сетки Сетки ввода строк
Ctrl+PgDn / Ctrl+PgUp Окончание ожидания прозвонки или соединения отображаемой линии (если в секции Main диалогового окна "Настройки" для этой функции не назначена клавиша Space) / Прибавление 30 сек. к значению времени ожидания Line / Flush Timeout, Add to Timeout
F1 / Alt+F1 Справочная система программы Радиус / Информация о программе Help / Help Topics, About
Ctrl+F3 Закрытие отображаемой линии почтовой системы Line / Close
F4 / Alt+F4 Вызов копии окна Радиуса / Закрытие текущего окна System / Mirror, Close Window
F5 / Shift+Ctrl+Alt+F5 / Ctrl+F5, Alt+F5 / Shift+F5 / Shift+Ctrl+F5 Создать Crash-прозвонку / Создать Normal-прозвонку / Удалить прозвонку / Сбросить счетчик неудачных прозвонок / Информация о статусе прозвонки / Удалить все прозвонки и файловые запросы Poll / Create Crash Poll, Create Normal Poll, Delete, Reset, Info, Delete All
F6 / Ctrl+F6 / Shift+F6 Открыть список узлов / Открыть список узлов по указанному адресу / Компилировать список узлов Tool/ Browse Nodelist, Browse Nodelist at..., Compile Nodelist
F7 Редактировать файловый запрос Tool/ Edit File Request...
F8 / Ctrl+F8 Отправить файлы на узел / Создать прозвонку типа Outb на узел Tool/ Attach Files..., Create a Flag
F9 Отправить команду модему Line / Send Modem Commands
Alt+F10 / Shift+F10 Вызов контекстного меню там, где оно вызывается правой кнопкой мыши Диалоговые окна
. (точка) Поставить на паузу / снять с паузы прозвонку Poll / Pause
Alt+0 Открытие/закрытие TCP/IP Daemon Line / TCP/IP Daemon
Alt+1 / Alt+2 / ... Alt+9 Открытие/закрытие первых 9 модемных линий Line / Dial-up Lines
Ctrl+A Ответ на входящий звонок Line / Answer Call
Ctrl+B Окно конфигурации файл-боксов Config / File-boxes
Ctrl+C Активизировать сеанс чата. Line / Open Chat
Ctrl+D Окно конфигурации модемных соединений Config / Dial-up
Ctrl+E Окно конфигурации адресатов шифрованного сеанса связи BinkP Config / Encrypted Links
Ctrl+F Окно конфигурации файловых запросов Config / File Requests
Ctrl+I Инспектор Узлов Config / Node Inspector
Ctrl+L / Ctrl+R Отложить текущий принимаемый файл до следующего сеанса / Отказаться от приема файла Line / Skip File, Reject File
Ctrl+N Окно конфигурации списка узлов Config / Nodelist
Ctrl+O Окно конфигурации параметров прозвонок Config / Polls
Ctrl+P Окно настройки каталогов Config / Paths
Ctrl+S Окно конфигурации параметров запуска Config / Start-up
Ctrl+T Окно конфигурации TCP/IP Daemon Config / TCP/IP Daemon
Ctrl+V Окно конфигурации событий Config / Events
Ctrl+W Окно конфигурации паролей Config / Passwords
Ctrl+X Окно конфигурации внешних приложений Config / Externals
Alt+X Выход из программы System / Exit
