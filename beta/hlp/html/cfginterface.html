Параметры интерфейса

Параметры этой секции касаются внешнего вида системы Радиус.

Параметр Описание
Interface Язык интерфейса, т.е. графической среды Радиуса: диалоговых окон, закладок линий и т.д. Доступны русский и английский языки.
Help Язык справочной системы Радиуса: русский или английский.
Use HTML help Используемый формат файла справки: старый, HLP, или новый, CHM (HTML).
Grid lines in BadWaZoo Показывать линии сетки в панели BadWaZOO.
Grid lines in Polls View Показывать линии сетки в менеджере прозвонок.
Font properties Эта кнопка позволяет настроить параметры шрифта окна хронологии.
Show icons in Main Menu Показывать значки иконок в главном меню.

Цвета

В этой подсекции располагаются настройки цветов фона и текста следующих элементов графической среды Радиуса: индикатор прогресса приема/передачи, окно хронологии, панель BadWaZoo. Кроме того, для удобства просмотра окна буфера отправки предусмотрена возможность подсветки различными цветами его строк в зависимости от даты создания неотправленных пакетов. Могут быть заданы отдельно цветовые параметры для пакетов возрастом старше: 7 дней; 14 дней; 21 дня; 28 дней.
