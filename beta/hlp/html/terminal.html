Терминал

В Радиусе имеется возможность запуска терминала. Терминал реализован в виде plug-in'а (подключаемого модуля), поставляемого с Пакетом утилит для Радиуса.

Для подключения терминала необходимо модуль radxterm.rdl из Пакета утилит для Радиуса скопировать в Домашний каталог Радиуса. После этого в главном меню становится доступным пункт Line / Terminal, при вызове которого активизируется окно терминала.
