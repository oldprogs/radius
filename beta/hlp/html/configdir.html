Каталог конфигурации

Configuration directory - это каталог, в котором хранятся бинарные файлы конфигурации Радиуса (config.bin, config.int, altcfg.bin, altcfg.int). Текстовый файл конфигурации (radius.ini) и файл конфигурации тарифов (tariff.cfg) находятся в Home Directory.

См. также Каталоги и Секция [Tariff].
