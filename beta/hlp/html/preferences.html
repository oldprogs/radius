Диалог "Настройки"

В это диалоговое окно вынесены элементы управления новыми возможностями Радиуса. Здесь задаются настройки визуального оформления Радиуса, системные параметры и многие дополнительные опции. Диалоговое окно содержит следующие секции:
  • Секция Main

  • Секция System

  • Секция Interface

  • Секция Tray

  • Секция Logs

  • Секция RCC

  • Секция RAS DialUp

  • Секция Protocols

  • Секция Netmail

  • Секция Tariff

  • Секция Sounds
