TAPI

TAPI (Telephony Application Programming Interface) - это стандартный программный интерфейс для создания приложений компьютерной телефонии в среде Windows, определяющий набор функций, которыми может пользоваться программа для взаимодействия с телефонами, линиями и коммутаторами. Стандарт TAPI является воплощением принципа открытости системной архитектуры и предназначен для разработки программ, ориентированных на рабочую станцию, выполняющую все интеллектуальные действия. Сам по себе интерфейс TAPI является аппаратно-независимым, что позволяет совместимым приложениям работать на различных машинах, используя разнообразные сетевые службы.

В процессе развития средств управления телефонными соединениями и распределения ресурсов между приложениями для персональных компьютеров потребовался общий интерфейс, позволяющий приложениям совместно использовать все доступные возможности. Кроме прочего, это означает, что способ обращения к данным должен быть идентичным для всех приложений. TAPI предоставляет простые и универсальные средства для управления соединениями между двумя или более компьютерами и для доступа к потокам данных, проходящим по этим соединениям. Управление средствами телефонии производится централизованно операционной системой, что позволяет обеспечить единый интерфейс для приложений, использующих на первый взгляд несовместимые протоколы обмена.

Установки Радиуса, касающиеся использования TAPI, производятся в окне настройки коммуникационных портов и в окне конфигурация модема.
