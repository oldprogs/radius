Что такое Радиус?

Радиус - это почтово-файловая транспортная система, предназначенная для многолинейной работы в сетях Dialup (коммутируемых модемных соединений) и TCP/IP (транспортного протокола сети Интернет) одновременно. Главная задача системы Радиус - прием и передача файлов в автоматическом режиме.

Радиус основан на коде программы Аргус 3.210. Возможности Аргуса в Радиусе сохранены и значительно расширены. Кроме того, исправлено множество ошибок, обнаруженных в процессе эксплуатации Аргуса.

Главными достоинствами системы Радиус являются:

  • Поддержка многопоточной работы Win32

  • Многолинейность

  • Поддержка TCP/IP

  • Возможность использования TAPI (new)

Характеристики системы Радиус:

  • Свободно распространяемое ПО, открытые исходные тексты
  • Работа под операционными системами Windows 9x / ME / NT / 2000 / XP, в том числе сервисом
  • BSO (Binkley-style outbound)
  • Расширения BSO: 5d BSO, динамический буфер отправки, динамическая маршрутизация личной почты (new)
  • Файл-боксы
  • Встроенный чат (new)
  • Подключаемый модуль терминала (new)
  • Поддержка файловых запросов: встроенная и с использованием SRIF
  • Прием факсов
  • Модуль расчета тарифов (new)
  • Консоль удаленного управления (new)
  • Богатые возможности для работы в автоматическом режиме: механизм обработки событий, запуск внешних приложений, поддержка скриптов для управления на основе анализа ответов модема (new), использование службы RAS для установки соединений с Интернетом (new)
  • Работа с файл-флагами
  • Обратный вызов (new)
  • Озвучивание событий (new)
  • Инспектор узлов - удобный инструмент настройки индивидуальных параметров узлов
  • Импорт списка узлов и файла паролей различных форматов
  • Поддержка регулярных выражений в масках
  • Сжатие исходящего потока средствами библиотеки zlib (new)
  • Возможность работы через прокси-сервер (new)
  • Шифрованный сеанс передачи и механизм шифрования паролей
  • Защита конфигурационных данных
  • И многое другое
