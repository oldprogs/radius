Соединения посредством RAS

Модемные соединения посредством RAS (Remote Access Server - сервер удаленного доступа) позволяют Радиусу подключаться к сети через модем. При установке соединения Радиус блокирует модемные линии, а после завершения сеанса связи вновь передает им управление модемом.

Настройка RAS-соединений производится установкой опции Enable RAS и выбором модемного соединения из числа созданных в Windows подключений к сети. Выбор модемного соединения производится из раскрывающегося списка Entry list. Кнопка Reload entrylist служит для ручного обновления списка доступных подключений.

После активизации RAS-соединений в нижней части окна TCP/IP Daemon появляется кнопка Connect, позволяющая запустить механизм соединения с сетью. Чтобы прервать процесс соединения, нажмите кнопку Cancel.

Для автоматизации соединения с сетью посредством RAS вы можете воспользоваться атомом RAS Event и макросами %STARTRAS%, %STOPRAS%.
