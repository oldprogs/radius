Конфигурация внешних приложений

Внешние приложения конфигурируются в диалоговом окне Config /Externals. Текущая версия системы Радиус поддерживает следующие типы внешних приложений:

  • Обработчики и сортировка

  • Внешние флаги

  • Сервисы

  • Периодически запускаемые приложения

  • Внешние утилиты

В нижней части описываемого диалогового окна расположена кнопка Run now, предназначенная для тестирования внешних приложений. Ее нажатие приведет к выполнению командной строки приложения, которому соответствует выделенное курсором поле сетки ввода.
