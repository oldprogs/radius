Текстовый файл конфигурации

Текстовый файл конфигурации radius.ini, находящийся в Домашнем каталоге, является одним из инструментов управления Радиусом. Однако поскольку параметры, задаваемые в нем, дублируют настройки, задаваемые из графической среды Радиуса (главным образом в диалоговом окне "Настройки", меню Config / Preferences), он может не использоваться вовсе или использоваться только для удаленного управления системой.

Файл конфигурации состоит из следующих секций:

  • Секция [Main]
  • Секция [System]
  • Секция [Polls]
  • Секция [Tray]
  • Секция [Interface]
  • Секция [Colors]
  • Секция [Sizes]
  • Секция [Tariff]
  • Секция [Remote]
  • Секция [Paths]
  • Секция [Hydra]
  • Секция [Binkp]
  • Секция [IP]
  • Секция [RasDial]
  • Секция [Netmail]
  • Секция [ExtApps]

Секции файла конфигурации приблизительно соответствуют закладкам диалогового окна "Настройки".
