Мастер запуска

При первом запуске Радиуса открывается диалоговое окно мастера запуска. Здесь вы можете настроить следующие параметры:

  • Для модемных соединений:

    • код области и телефонные префиксы для локальных и междугородних звонков

    • тип набора (тоновый или импульсный)

    • номер COM-порта

    • автоматическое формирование схемы ограничений, предотвращающей междугородние звонки

  • Для TCP/IP-соединений:

    • возможность автоматического запуска TCP/IP-демона
