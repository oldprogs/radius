Меню Help

  • Help Topics - при выборе этого пункта вызывается справочная система Радиуса, с которой Вы в данный момент работаете.

  • Argus Related Web Sites - посетить один из Web-серверов, связанных с системой Радиус: сервер поддержки Аргуса, сервер поддержки Радиуса, общий сервер поддержки.

  • About - при выборе этого пункта выводится информационное окно, содержащее сведения о версии программы, авторских правах и т.д.
